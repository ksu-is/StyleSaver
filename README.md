## Project Overview
A program designed to be your smart shopping assistant that helps users stay stylish while saving money. It does this by recommending personalized fashion items while tracking your spending and giving budget-friendly suggestions.

---

## Project Goals
- Help users manage their shopping budget  
- Provide real-time spending tracking and alerts  
- Offer AI-based fashion recommendations  
- Create visual charts to show monthly spending patterns  
- Support budgeting education through easy-to-use analytics
